# Odin Project - Landing Page

A responsive landing page built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Overview

This project is a recreation of a landing page following the specifications provided by The Odin Project. The goal was to practice building a complete, visually appealing webpage using HTML and CSS, with a strong focus on **Flexbox** for layout.

## Project Structure

## Features

- **Semantic HTML5** structure using `<header>`, `<nav>`, `<section>`, `<blockquote>`, etc. for better readability and accessibility.
- **Flexbox** layout throughout the entire page for responsive and clean alignment.
- **Hover effects** on the main content images:
  - Images pop out of their blue frames.
  - Smooth scaling/enlargement using CSS `transform` and `transition`.
- Clean navigation bar.
- Hero section with title, short description, and hero image.
- Image grid with accompanying text descriptions.
- Inspirational quote section.
- Footer with attribution.

## Technologies Used

- HTML5
- CSS3 (Flexbox, hover states, transitions)
